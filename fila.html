리액트 기반 반응형 쇼핑몰 홈페이지 개발
React-Bootstrap을 사용한 반응형 쇼핑몰 홈페이지로 검색 기능과 로그인시 상세 페이지가 보여지도록 개발하였습니다.

Overview

React-Router 라이브러리와 React-Bootstarp을 사용한 리액트 SPA 홈페이지를 개발하였습니다.
데이터를 json 파일로 정리하였고, json-server 라이브러리 를 통해 짧은 시간에 REST API 를 구축하였습니다.
