문화 공연 예매 플랫폼 웹/앱 사이트 개발 (PWA)
Scss, Javascript, jQuery를 중점으로 애니메이션을 활용하여 조금 더 동적인 효과를 구현하여 사이트를 개발하였습니다.

Overview

기존의 문화 공연 플랫폼 <매표소> 의 경우 동일한 사이즈의 이미지 배열로 되어 있어 사용자의 흥미를 끌기 어려울 것이라고 판단하였습니다.
이러한 단점을 보완하여 사용자의 이목을 끈느 것과 동시에 더욱 편하게 서비스를 이용할 수 있도록 리디자인하여 개발하였습니다.
이미지는 Swiper API, Isotope Plugin를 적용하여 보다 액티비티한 효과를 구현하고자 하였습니다.
애플리케이션을 설치하지 않더라도 사용자의 웹 사이트 방문이 용이하도록 PWA 웹 앱 사이트를 구축하였습니다.
