Overview

Vue를 이용한 < To Do List > 웹 사이트를 개발하였습니다.
To do 추가/제거시 Vue TransitionGroup을 통해 동적인 리스트 효과를 주었습니다.
리스트의 우측 쓰레기통 아이콘 클릭하여 개별 리스트를 삭제할 수 있고, 하단의 Clear All 버튼을 클릭하면 전체 리스트가 삭제되도록 하였습니다.
내용을 입력하지 않고 추가 버튼을 눌렀을 경우 Vue modal을 이용하여 경고 alert 창이 나오도록 구현하였습니다.
